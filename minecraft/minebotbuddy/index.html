MineBotBuddy

MineBotBuddy is an acessibility mod for Minecraft. MineBotBuddy takes over a Minecraft client and can be controlled using various methods.

Chat mode

Chat mode allows the bot to be controlled using Minecrat's in game chat and only requires the mod to be installed on the bot's client. The only configuration this mode requires is to set the usernames of the people who are allowed to control the bot.

Voice mode

Voice mode allows the bot to be controlled by voice commands. This mode requires the mod to be installed on both the bot's client and the player's client. Additional configuration is required to establish a connection between the player's and bot's clients.
